The Phases of Agile Development


Source



Phase 0: Requirements Analysis


Meet with product owner to create user stories

Phase 1: Planning


Meet with product owner to determine which user stories will be completed each sprint

Phase 2: Design


Identify ways to implement and complete the tasks required for the sprint

Phase 3: Build


Implementing the ways identified in the previous phase

Phase 4: Testing


Ensure system functionality and debugging

Phase 5: Release (Current Project Phase)


Presenting the shippable pieces of the product to the product owner

Phase 6: Track and Monitor


Continue to provide updates to the product over time
